# POS Menu Digital

Aplikasi Point of Sale (POS) dan Menu Digital untuk restoran/kantin dengan fitur:
- Menu Digital (Scan QR Code)
- Sistem Kasir Real-time
- Manajemen Menu & Kategori
- Laporan Penjualan (CSV)
- Sistem Cashbon (Belum Bayar)

## Cara Menjalankan (Lokal)
1. Clone repository ini.
2. Jalankan `npm install`.
3. Jalankan `npm run dev`.
4. Buka `http://localhost:3000`.

## Deployment
Aplikasi ini sudah dikonfigurasi untuk di-deploy ke **Railway.app** (dengan database permanen) atau **Vercel**.
